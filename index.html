üìã Class - Blueprint/Template

Class adalah cetak biru untuk membuat objek. Seperti desain rumah sebelum dibangun.

// Contoh Class Mobil
class Mobil {
    // Attribute
    merek;
    warna;
    tahun;
    
    // Constructor
    constructor(merek, warna, tahun) {
        this.merek = merek;
        this.warna = warna;
        this.tahun = tahun;
    }
    
    // Method
    info() {
        return `Mobil ${this.merek} warna ${this.warna} tahun ${this.tahun}`;
    }
}

üîí Encapsulation - Pengamanan Data

Encapsulation menyembunyikan data sensitif, hanya bisa diakses lewat method tertentu.

Contoh ini diambil dari kehidupan sehari-hari seperti ATM, di mana saldo tidak bisa diakses langsung tanpa proses yang benar.

class RekeningBank {
    #saldo; // Private attribute (pakai #)
    
    constructor(saldoAwal) {
        this.#saldo = saldoAwal;
    }
    
    // Getter - untuk melihat saldo
    getSaldo() {
        return this.#saldo;
    }
    
    // Method untuk deposit
    deposit(jumlah) {
        if (jumlah > 0) {
            this.#saldo += jumlah;
            return `Deposit berhasil! Saldo: Rp ${this.#saldo.toLocaleString()}`;
        }
        return "Jumlah harus positif!";
    }
    
    // Method untuk tarik tunai
    tarikTunai(jumlah) {
        if (jumlah > this.#saldo) {
            return "Saldo tidak cukup!";
        }
        this.#saldo -= jumlah;
        return `Penarikan berhasil! Saldo: Rp ${this.#saldo.toLocaleString()}`;
    }
}

üë®‚Äçüë¶ Inheritance - Pewarisan

Class anak mewarisi sifat dan method dari class induk.

// Class Induk
class Manusia {
    constructor(nama, umur) {
        this.nama = nama;
        this.umur = umur;
    }
    
    perkenalan() {
        return `Halo, nama saya ${this.nama}, umur ${this.umur} tahun`;
    }
}

// Class Anak mewarisi dari Manusia
class Mahasiswa extends Manusia {
    constructor(nama, umur, nim, jurusan) {
        super(nama, umur); // Memanggil constructor induk
        this.nim = nim;
        this.jurusan = jurusan;
    }
    
    belajar() {
        return `${this.nama} sedang belajar di jurusan ${this.jurusan}`;
    }
}
